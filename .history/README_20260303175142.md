# AI Stress Intelligence System

## Overview
AI Stress Intelligence System is a hybrid machine learning + local LLM-powered stress analysis platform.

The system combines:
- Routine behavioral inputs
- Live facial analysis
- Random Forest stress classification
- SHAP explainability
- Local LLaMA3 reasoning layer
- Personalized AI-generated PDF reports

## Features

- Live facial scanning with OpenCV
- Behavioral + emotional stress modeling
- Explainable AI influence breakdown
- Personalized AI cognitive report
- PDF downloadable report
- Fully offline local AI (Ollama LLaMA3)

## Architecture

Routine Data + Facial Features  
→ Random Forest Stress Classifier  
→ SHAP Feature Contribution  
→ LLaMA3 Cognitive Reasoning  
→ Personalized AI Report + PDF

## Installation

1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Install Ollama and pull model:
   ollama pull llama3
4. Run:
   streamlit run src/ui/app_streamlit.py

## Technologies Used

- Python
- Streamlit
- OpenCV
- Scikit-learn
- SHAP
- ReportLab
- Ollama (LLaMA3)
