# AI Stress Intelligence System  
### Hybrid Machine Learning + Local LLM-Based Stress Analysis Platform

---

## 📌 Overview

AI Stress Intelligence System is a multi-modal hybrid AI platform designed to analyze and interpret human stress levels using:

- Behavioral routine data  
- Real-time facial analysis  
- Machine learning classification  
- Explainable AI (SHAP - SHapley Additive exPlanations)  
- Local Large Language Model reasoning (LLaMA3)  
- Personalized AI-generated PDF reports  

The system acts as a **decision-support tool** and does not provide medical diagnosis.  
It is developed for academic and research purposes in affective computing and behavioral analytics.

---

## 🎯 Objectives

- Detect stress levels: **Low / Moderate / High**
- Combine objective facial indicators with subjective routine inputs
- Provide explainable predictions
- Generate personalized AI cognitive reports
- Maintain ethical and privacy-aware design
- Operate fully offline using local AI

---

## 🧠 Core Components

### 1️⃣ Behavioral Feature Input

User-provided routine data:

- Sleep Hours
- Sleep Quality
- Workload Hours
- Assignment Pressure
- Physical Activity Duration
- Screen Time
- Social Interaction
- Caffeine Intake

---

### 2️⃣ Facial Feature Extraction (Live Capture)

Using OpenCV + MediaPipe:

- Eye Opening Ratio
- Mouth Ratio
- Emotion Probabilities:
  - Angry
  - Fear
  - Sad
  - Neutral
  - Happy
  - Surprise

Facial signals are captured over a session window and averaged for stability.

---

### 3️⃣ Stress Scoring Logic (Synthetic Dataset Design)

Stress labels were generated using a weighted behavioral scoring model:

```
Stress Score = Weighted Cognitive Load Factors
          + Weighted Recovery Factors
          + Emotional Stress Indicators
          + Controlled Noise
```

Stress classes:
- `0` → Low
- `1` → Moderate
- `2` → High

**Dataset:**
- 5000 balanced samples
- Multi-class classification
- Validated for:
  - Range correctness
  - Missing values
  - Class balance
  - Emotion normalization

---

### 4️⃣ Machine Learning Model

**Models evaluated:**
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest (Final Selected)

**Final Model:**
- Random Forest Classifier
- Feature scaling via StandardScaler
- 80–20 train-test split
- Evaluated using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - Confusion Matrix

---

### 5️⃣ Explainable AI (SHAP)

SHAP (SHapley Additive exPlanations) is used for:

- Local feature contribution analysis
- Identifying top influencing factors per prediction
- Increasing model transparency

---

### 6️⃣ Local LLM Reasoning Layer

Integrated via Ollama:

- **Model:** `llama3`
- **Fully offline inference**
- Converts structured ML output into:
  - Human-readable stress interpretation
  - Behavioral insights
  - Structured recommendations
- Personalized greeting using user name

*Note: LLM interprets ML results; it does not replace the classifier.*

---

### 7️⃣ Frontend (Streamlit)

Features:

- High-tech dark UI theme
- Stress gauge visualization
- Facial signal summary
- AI Cognitive Report section
- Personalized PDF download

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone <https://github.com/PratapOP/Stress_Management_System>
cd Stress_Management_System
```

### 2️⃣ Create Virtual Environment

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/Mac:**
```bash
python -m venv venv
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Install Ollama & Pull LLaMA3

```bash
ollama pull llama3
```

Ensure Ollama server is running:

```bash
ollama serve
```

---

## ▶️ Run the Application

```bash
streamlit run src/ui/app_streamlit.py
```
